Processo

Como usar o kit

Cada template é independente. Use na ordem que fizer sentido para o seu momento.

01
Escolha o template
Cada card cobre uma etapa do ciclo de produto — do discovery à entrega. Abra o que você precisa agora.
02
Preencha os campos
Os campos têm dicas de como preencher. Use os exemplos como referência e adapte para o contexto do seu produto.
03
Use a IA para acelerar
Cada template tem uma dica de IA com um prompt pronto. Cole no Claude ou ChatGPT e ganhe velocidade.
